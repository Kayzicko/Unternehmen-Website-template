# Unternehmen-Website Template

Diese HTML-Vorlage eignet sich perfekt für Coaches, Dienstleister oder Selbstständige.

## 🔧 Anleitung

Öffne die Datei `index.html` und ersetze die Platzhalter in `[[DOPPELKLAMMERN]]`:

- [[NAME]] → Dein Name
- [[BERUF]] → z. B. Personal Trainer, Coach, Berater
- [[EMAIL]] → deine Mailadresse
- [[TELEFON]] → Telefonnummer im Format +49...
- [[PROFILBILD-PFAD]] → z. B. images/profilbild.png
- [[ERFAHRUNG]] → z. B. „seit 2019“

Speichere und öffne die Datei im Browser – fertig ist deine persönliche Website!

## 🌍 Hosting-Tipp

Du kannst die Seite kostenlos bei GitHub Pages oder Netlify veröffentlichen.
